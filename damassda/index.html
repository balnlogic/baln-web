How it works

3단계로 끝나는 장보기

복잡한 계산기 앱 필요 없이, 카메라 한 번이면 됩니다.

1
📸

가격표 촬영

카메라로 가격표를 찍으면
AI가 텍스트를 자동 인식합니다.

2
🏷️

상품 자동 등록

상품명과 가격을 자동 추출.
할인가가 있으면 할인가를 적용합니다.

3
🧮

실시간 합계

담은 상품의 합계를 실시간으로 확인.
예산 초과 시 경고해줍니다.

Features

장보기가 편해지는 기능들

가격 인식부터 예산 관리까지, 장보기에 필요한 것만 담았습니다.

📷

가격표 OCR 인식

카메라로 가격표를 촬영하면 온디바이스 AI가 상품명과 가격을 자동으로 읽어냅니다. 인터넷 없이도 동작합니다.

💰

할인가 자동 적용

할인, 행사, 세일 등 할인 키워드를 감지하면 자동으로 할인가를 우선 적용합니다.

📊

실시간 예산 관리

설정한 예산 대비 현재 합계를 실시간으로 보여주고, 초과 위험 시 단계별로 경고합니다.

⌨️

빠른 수동 입력

가격표가 인식되지 않을 때는 하단 키패드로 직접 가격을 입력할 수 있습니다.

↩️

실수 방지

잘못 담은 상품은 실행 취소 토스트로 바로 되돌릴 수 있습니다.

🎁

1+1 / 2+1 할인

같은 상품 2개 이상 담으면 1+1, 2+1 행사 적용을 자동으로 제안합니다.

🔒

완전한 오프라인, 완전한 프라이버시

모든 가격 인식은 기기 내에서 처리됩니다. 서버에 사진을 보내지 않고, 장보기 데이터는 기기에만 저장됩니다. 인터넷이 안 되는 마트 지하에서도 문제없이 동작합니다.

온디바이스 AI 오프라인 동작 데이터 기기 저장 서버 전송 없음
