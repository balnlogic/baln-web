How it works

감지 → 진동 → 멈춤, 3단계

팔꿈치로 찔리는 밤은 이제 끝. Watch가 조용히 해결합니다.

1
🎙️

코골이 감지

Apple Watch 마이크가 수면 중
코골이 소리를 실시간 감지합니다.

2
📳

3단계 진동

약한 진동 → 강한 진동 → iPhone 진동.
대부분 1단계에서 자세 변화로 멈춥니다.

3
📊

아침 리포트

어젯밤 코골이 횟수, 진동 반응률,
수면 품질 점수를 아침에 확인합니다.

Features

녹음 앱과는 다릅니다

코골이를 기록만 하지 않고, 실제로 해결합니다.

⌚

Apple Watch 실시간 감지

Watch 마이크로 코골이를 실시간 모니터링. 배경 소음과 구별하여 정확하게 감지합니다.

📳

3단계 에스컬레이션 진동

약한 진동(5초) → 강한 진동(15초) → iPhone 진동(30초). 부드럽게 자세 변화를 유도합니다.

🧠

AI 패턴 분석

음주, 운동, 스트레스 등 생활습관과 코골이의 상관관계를 자동 분석합니다.

💯

수면 품질 점수

코골이 횟수, 진동 반응률, 수면 시간을 종합하여 0~100점 수면 품질 점수를 제공합니다.

💑

파트너 공유

코골이 개선 현황을 파트너에게 공유. 옆 사람에게 "나 진짜 노력하고 있어"를 증명합니다.

⏰

스마트 알람

수면 데이터를 분석하여 가벼운 수면 구간에 깨워줍니다. 개운한 아침을 경험하세요.

🔒

100% 로컬 처리, 0% 서버 전송

수면 중 녹음, 코골이 데이터, 건강 정보 모두 기기 안에서만 처리됩니다. 외부 서버에 어떤 데이터도 전송하지 않습니다. 광고도, 추적도 없습니다.

온디바이스 처리 서버 전송 없음 광고 없음 추적 없음
