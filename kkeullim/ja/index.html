Features

スペックではなく、ライフスタイルで

年収や学歴の代わりに、フィットネスとセルフケアで魅力を証明します。

📊

InBody認証

体組成データを提出すると運営チームが審査。実際に運動している人だけが認証バッジを獲得します。

🤳

リアルタイムセルフィー認証

リアルタイム撮影でプロフィール写真とAI照合。なりすましを根本的にブロックします。

📸

ボディプロフィールギャラリー

トレーニングの成果を堂々と公開。ボディプロフィール、ワークアウト写真、OOTDを共有。

📞

チャットなしで直接連絡

マッチ即電話番号公開。アプリ内の気まずいチャットなしですぐ会う約束ができます。

🚫

知人ブロック

連絡先を同期すると知人を自動ブロック。SHA-256ハッシュで電話番号の原本は保存しません。

🛡️

3段階安全システム

セルフィー認証 + 不適切コンテンツ自動検出 + 24時間モニタリング。偽プロフィールをブロック。

🔒

個人情報、鉄壁保護

認証完了後、原本書類は即座に永久削除。バッジのみ保存します。写真アップロード時にGPS・デバイス情報(EXIF)を自動削除し、プロフィール画面はスクリーンショットがブロックされます。

認証書類即削除 EXIF自動削除 スクリーンショットブロック 連絡先ハッシュ保存
