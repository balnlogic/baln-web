닥체크
병원 비교 & 과잉진료 체크 · DocCheck

닥체크는 사용자의 개인정보를 수집하지 않습니다. 로그인도 필요 없습니다.

1. 개인정보 처리방침 안내

닥체크(이하 "앱")는 baln 팀이 운영하는 병원 비교 및 과잉진료 정보 제공 서비스입니다. 본 방침은 앱 이용 과정에서 사용자의 개인정보를 어떻게 처리하는지 설명합니다.

시행일: 2026년 3월 14일

2. 수집하는 개인정보

수집 없음 (Data Not Collected)

닥체크는 현재 어떠한 개인정보도 수집하지 않습니다. 회원가입이나 로그인 없이 모든 기능을 사용할 수 있습니다.

항목 수집 여부 비고
이름, 이메일, 전화번호 수집 안 함 계정 불필요
위치 정보 (GPS) 수집 안 함 향후 선택적 기능 추가 시 별도 동의
기기 식별자 수집 안 함 —
검색 기록 외부 전송 없음 기기 내 임시 캐시만 저장, 앱 삭제 시 소멸
분석 데이터 (Analytics) 수집 안 함 제3자 분석 SDK 미사용
광고 식별자 (IDFA) 수집 안 함 광고 없음

3. 데이터 출처

닥체크가 표시하는 모든 병원 정보 및 통계는 아래 공공 기관의 공개 데이터를 사용합니다. 사용자의 개인 의료 정보와는 무관합니다.

  • 건강보험심사평가원(HIRA) 공개 데이터 포털
  • 국민건강보험공단 공개 통계
  • 보건복지부 의료기관 공개 정보

4. 제3자 정보 제공

닥체크는 사용자 정보를 제3자에게 제공하지 않습니다. 수집하는 개인정보 자체가 없으므로 제공할 정보도 존재하지 않습니다.

5. 쿠키 및 추적 기술

앱은 쿠키, 광고 추적, 크로스앱 추적 기술을 사용하지 않습니다. Apple의 앱 추적 투명성(ATT) 프레임워크에 따라 사용자 추적을 일절 하지 않습니다.

6. 아동 개인정보 보호

닥체크는 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 앱은 개인정보를 수집하지 않으므로 아동도 안전하게 이용할 수 있습니다.

7. 보안

앱 내에서 처리되는 데이터는 기기를 벗어나지 않으며, 외부 서버 전송 시 HTTPS(TLS) 암호화 통신을 사용합니다. 전송되는 데이터는 병원 정보 조회 요청뿐이며, 개인 식별 정보를 포함하지 않습니다.

8. 방침 변경

개인정보처리방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 사전 고지합니다. 중요한 변경 사항이 있을 경우 앱 내 공지를 통해 별도 안내합니다.

의료 정보 면책 고지
닥체크에서 제공하는 모든 정보는 공개된 통계 데이터를 기반으로 한 참고 자료입니다. 어떠한 정보도 의학적 조언, 진단, 치료 권고를 대체하지 않습니다. 건강 관련 결정은 반드시 전문 의료인과 상담하시기 바랍니다.
