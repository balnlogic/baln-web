Features

스펙이 아닌, 라이프스타일

연봉이나 학력 대신, 자기관리와 라이프스타일로 매력을 증명합니다.

💪

InBody 인증

체성분 결과지 제출 후 운영팀 검수. 실제 운동하는 사람에게 인증 뱃지를 부여합니다.

📷

실시간 셀피 인증

실시간 촬영으로 AI가 프로필 사진 대조. 사진 도용을 원천 차단합니다.

🔥

OOTD 피드

오늘의 운동 룩을 공유하고, 운동 인증샷으로 일상을 보여주세요.

💬

매칭 즉시 채팅 + 연락처

매칭되면 1:1 채팅이 열리고 연락처가 공개됩니다. 바로 대화하고 만나세요.

🚫

지인 차단

연락처 동기화로 지인 자동 차단. SHA-256 해시로 번호 원본은 저장하지 않습니다.

🔐

스크린샷 차단

프로필 화면 스크린샷 방지. 사진 업로드 시 GPS/기기 정보(EXIF) 자동 제거.

🏋

운동 인증

헬스장 회원권, PT 영수증, 크로스핏 등록증으로 자기관리를 증명합니다.

📈

운동 기반 매칭

운동 종목, 빈도, 시간대, 목표까지. 같은 라이프스타일의 사람을 만나세요.

🛡

개인정보, 철벽 보호

인증 서류는 검수 후 즉시 영구 삭제하고 배지만 남깁니다. 매칭 전까지 연락처는 절대 노출되지 않습니다.

인증 서류 즉시 삭제 EXIF 자동 제거 스크린샷 차단 연락처 SHA-256 해시 매칭 전 연락처 비공개
