Features

스펙이 아닌, 라이프스타일로

연봉·학력 검증 대신 운동과 자기 관리로 매력을 증명합니다.

📊

InBody 인증

체성분 결과지를 제출하면 운영팀이 검수. 실제 운동하는 사람인지 검증된 배지를 획득합니다.

🤳

실시간 셀피 인증

실시간 촬영으로 프로필 사진과 AI 대조. 사진 도용을 원천 차단합니다.

📸

바디프로필 갤러리

운동 결과물을 당당하게 공개. 바디프로필, 운동 인증샷, OOTD를 공유하세요.

📞

채팅 없이 바로 연락

골드스푼 모델: 매칭 즉시 전화번호 공개. 앱 안에서 어색한 채팅 없이 바로 만남.

🚫

지인 차단

연락처를 동기화하면 지인을 자동 차단. SHA-256 해시로 번호 원본은 저장하지 않습니다.

🛡️

3단계 안전 시스템

셀피 인증 + 부적절 콘텐츠 자동 감지 + 24시간 모니터링. 허위 프로필을 차단합니다.

🔒

개인정보, 철벽 보호

인증 완료 후 원본 서류는 즉시 영구 삭제. 배지만 저장합니다. 사진 업로드 시 GPS·기기 정보(EXIF)를 자동 제거하고, 프로필 화면은 스크린샷이 차단됩니다.

인증 서류 즉시 삭제 EXIF 자동 제거 스크린샷 차단 연락처 해시 저장
