How it works

감지 → 진동 → 멈춤, 3단계

팔꿈치로 찔리는 밤은 이제 끝. Watch가 조용히 해결합니다.

1
🎙️

코골이 감지

Apple Watch 마이크가 수면 중
코골이 소리를 실시간 감지합니다.

2
📳

3단계 진동

약한 진동 → 강한 진동 → iPhone 진동.
대부분 1단계에서 자세 변화로 멈춥니다.

3
📊

아침 리포트

옆사람 방해 점수, 소리별 녹음 재생,
생활습관 상관관계까지 아침에 확인합니다.

Features

녹음 앱과는 다릅니다

코골이를 기록만 하지 않고, 실제로 해결합니다.

😴

옆사람 방해 점수

소음 노출 시간, 강도별 빈도, 최대 데시벨, 연속 지속 시간을 종합해 옆 사람이 얼마나 힘들었는지 0~100점으로 보여줍니다.

🎙️

코골이 · 기침 · 잠꼬대 분류

ML 기반으로 소리 종류를 자동 구분하고, 각각 녹음 파일로 저장합니다. 타임라인에서 탭하면 바로 재생됩니다.

📳

3단계 에스컬레이션 진동

약한 진동 → 강한 진동 → iPhone 진동. 부드럽게 자세 변화를 유도합니다. 대부분 1단계에서 멈춥니다.

📊

생활습관 상관관계 분석

음주, 운동, 커피, 스트레스, 체중과 코골이의 관계를 14일간 추적해 "음주한 날 방해 점수가 23점 높았어요" 같은 인사이트를 제공합니다.

⌚

Watch + iPhone 듀얼 모드

Apple Watch로 감지하고 iPhone에서 분석 · 녹음 · 저장. Watch 없이 iPhone 단독 모드도 지원합니다.

💑

파트너 공유

방해 점수와 개선 현황을 파트너에게 공유. 옆 사람에게 "나 진짜 노력하고 있어"를 증명합니다.

⏰

스마트 알람

수면 데이터를 분석하여 가벼운 수면 구간에 깨워줍니다. 개운한 아침을 경험하세요.

🔒

100% 로컬 · 100% 무료

모든 데이터는 기기 안에서만 처리됩니다. 서버 전송, 광고, 추적 없음. 유료화 계획도 없습니다.

📈

숫자 하나가 행동을 바꿉니다

"코골이 87회"는 알아도 뭘 해야 할지 모릅니다. 하지만 "어젯밤 옆사람 방해 점수 72점 — 심함"을 보면 달라집니다. 생활습관 체크인과 함께 어떤 날 코골이가 심한지 패턴을 찾아보세요.
