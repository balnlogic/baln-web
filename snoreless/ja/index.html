How it works

検知 → 振動 → 停止、3ステップ

肘でつつかれる夜はもう終わり。Watchが静かに解決します。

Features

録音アプリとは違います

いびきを記録するだけでなく、実際に解決します。

😴

パートナー迷惑スコア

騒音露出時間、強度別頻度、最大デシベル、連続持続時間を総合し、パートナーへの影響を0〜100点で表示します。

🎙️

いびき・咳・寝言を分類

ML基盤で音声タイプを自動分類し、それぞれ録音ファイルとして保存。タイムラインでタップすれば即再生できます。

📳

3段階エスカレーション振動

弱い振動 → 強い振動 → iPhoneアラート。やさしく姿勢変化を促します。ほとんどが1段階で止まります。

📊

生活習慣相関分析

飲酒、運動、コーヒー、ストレス、体重といびきの関係を14日間追跡。「飲酒した日は迷惑スコアが23点高い」などのインサイトを提供します。

⌚

Watch + iPhone デュアルモード

Watchで検知しiPhoneで分析・録音・保存。Watch無しのiPhone単独モードも対応しています。

💑

パートナー共有

迷惑スコアと改善状況をパートナーに共有。「ちゃんと努力してるよ」を証明できます。

⏰

スマートアラーム

睡眠データを分析し、浅い眠りの段階で起こします。すっきりした朝を体験してください。

🔒

100%ローカル・100%無料

すべてのデータはデバイス内で処理。サーバー送信、広告、トラッキングなし。有料化の予定もありません。

📈

ひとつの数字が行動を変える

「いびき87回」を知っても、何をすべきかわかりません。 でも「パートナー迷惑スコア72点 — 重度」を見れば変わります。 毎日のチェックインと合わせて、どんな習慣がいびきを悪化させるかパターンを見つけましょう。
